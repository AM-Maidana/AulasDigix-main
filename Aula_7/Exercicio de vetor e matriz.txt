1. Escreva um programa que peça ao usuário para inserir 10 números em um vetor e
calcule a média desses valores.

2. Dada uma matriz 4x4 preenchida com valores, some os elementos de cada linha e
exiba o resultado.

3. Dado um vetor de inteiros, verifique se ele é um palíndromo (ou seja, se é igual
quando lido de trás para frente).

4. Dada uma matriz quadrada n x n, exiba a soma dos elementos da diagonal
principal e da diagonal secundária.

5. Crie um programa para rotacionar uma matriz n x n 90 graus no sentido horário.
